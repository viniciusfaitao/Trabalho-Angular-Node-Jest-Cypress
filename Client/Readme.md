# Client
`Infomações para utilizar a aplicação`

## Instalando e executando aplicação

Esteja na pasta principal

### `cd /helping-pets/client`

Instale as dependencias

### `npm install`

Execute a aplicação

### `npm start`

## Testando aplicação com Cypress

Esteja na pasta principal

### `cd /helping-pets/client`

Inicie o ambiente de test

### `npm run e2e`
